## Title

CleanText Studio — A privacy-first AI text cleanup and document formatting tool

## Short description

CleanText Studio cleans copied and AI-assisted text locally, restores document structure, preserves supported LaTeX formulas, and exports polished DOCX or TXT files for Windows.

## Announcement

Messy copied text should not force you to rebuild a document by hand. CleanText Studio is a local-first Windows desktop app for cleaning Markdown residue, recovering headings/lists/tables, preserving supported formulas, and exporting editable Word documents.

What it offers:

- Local cleanup and preview by default
- Optional user-owned API configuration for AI refinement
- Structured Word tables and editable OMML equations
- Runtime multilingual UI and light/dark themes
- Windows installer and portable package through GitHub Releases

See the screenshots and installation guidance in the repository README. Feedback, focused bug reports, translation review, and contributions are welcome.

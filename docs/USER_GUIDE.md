# 用户指南

启动后在左侧粘贴文本或打开 TXT、MD、DOCX，在中间选择预设、换行等级和列表处理方式，再点击“开始清洗”。原文始终保留，右侧结果可编辑、撤销、恢复、复制及导出。修改原文后会提示结果过期，修改结果后会标记手动编辑。聊天式套话清理默认关闭。TXT 默认 UTF-8；Word 使用本地排版模板。应用不联网、不上传文本、不收集遥测。

“段落与换行”提供删除段落间换行、仅保留大分段之间的换行（默认）和保留所有段落间换行。聊天式套话清理只处理首尾明确框架语，默认关闭。

AI 智能优化是可选步骤。先完成本地清洗，再在设置中配置自己的 Provider、Base URL、Model 和 API Key。点击 AI 智能优化后检查发送字符数、分块数、敏感信息和费用提示；AI 建议不会自动覆盖本地结果。无 API 配置或无网络时，本地清洗和导出不受影响。

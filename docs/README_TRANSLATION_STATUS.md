# README translation status

The v1.5.1 repository contains a complete README homepage for each supported
documentation locale: English, Simplified Chinese, Traditional Chinese,
Japanese, Korean, Spanish, French, German, Brazilian Portuguese, Russian,
Arabic, and Hindi.

The English README is the factual source. `docs/readme/build_readmes.py`
protects URLs, commands, file names, screenshots, and brand terms while
creating a full localized Markdown draft. Each committed README is ordinary
Markdown and can be read on GitHub without running the generator.

The project welcomes community review of terminology, clarity, and regional
usage. This status does not claim that every documentation page has received a
native-speaker editorial review.

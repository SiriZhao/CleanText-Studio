# Product Hunt launch material

## Product name

CleanText Studio

## Tagline

**Clean AI-generated text into professional documents locally.**

## Description (under 500 characters)

CleanText Studio is a privacy-first Windows desktop app for cleaning copied and
AI-assisted text. It removes Markdown residue, restores headings, lists,
tables, links, and supported LaTeX formulas, then exports polished DOCX or TXT
files. Basic cleanup stays local. Optional AI refinement uses your own provider
and API key.

## Maker introduction

I built CleanText Studio after repeatedly rebuilding structure by hand when
moving notes and AI-assisted drafts into Word. The aim is a transparent desktop
workflow: keep ordinary cleanup local, show the result before export, and keep
the source open for review and improvement.

## First comment

Hi Product Hunt — I made CleanText Studio for the awkward last mile between a
messy copied draft and a document someone can actually edit. The baseline
workflow is local: paste text, choose cleanup rules, inspect the result, and
export DOCX/TXT. AI refinement is optional and only calls a provider the user
configures.

The project is open source, and I would appreciate specific feedback on the
DOCX output, formula support, and the local-first data boundary. I will answer
questions candidly, including limitations.

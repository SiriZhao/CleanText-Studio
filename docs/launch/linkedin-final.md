# LinkedIn announcement draft

CleanText Studio is now available as an open-source, local-first Windows tool
for turning copied and AI-assisted text into structured DOCX or TXT documents.
It preserves useful headings, lists, tables, links, and supported equations so
document cleanup does not become a manual rebuild.

Normal cleanup and export run locally. Optional AI refinement is explicit and
uses the provider/API key selected by the user. The project is looking for
practical feedback on DOCX fidelity, table fixtures, localization review, and
Windows packaging.

Repository: https://github.com/SiriZhao/CleanText-Studio

# AlternativeTo submission copy

## Summary

CleanText Studio is a privacy-first Windows desktop tool for cleaning copied and
AI-assisted text into structured DOCX or TXT documents. It removes formatting
residue while preserving headings, lists, tables, links, and supported
equations. Standard cleanup runs locally; optional AI refinement uses a
provider/API key chosen by the user.

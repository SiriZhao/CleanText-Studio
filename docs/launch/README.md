# CleanText Studio open-source launch kit

This directory contains source copy and checklists for sharing CleanText Studio
with technical communities. It is not an automation guide and does not ask for
coordinated voting, artificial engagement, or undisclosed promotion.

## Positioning

> **CleanText Studio is a privacy-first, local-first AI text cleanup and document formatting application.**

It turns copied and AI-assisted text into clean, structured documents while
preserving useful headings, lists, tables, links, and supported LaTeX formulas.
Basic cleanup and export run locally; AI refinement is optional and uses a
provider/API key chosen by the user.

## Before publishing

1. Verify the latest tagged Windows assets and SHA256 file on the [Releases page](https://github.com/SiriZhao/CleanText-Studio/releases).
2. Confirm screenshots still match the released UI and contain no credentials or private paths.
3. Read each community's current self-promotion rules before posting.
4. Use one honest, audience-specific post—not copy-pasted announcements across communities.
5. Reply to technical questions, limitations, and bug reports candidly.

## Publication principles

- Describe current capabilities rather than promises.
- Keep every announcement focused on document quality, local processing, and developer feedback.
- Do not solicit votes or Stars as a condition of support.
- Treat feedback as product research, not a conversion funnel.

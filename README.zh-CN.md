# 净文排版 · CleanText Studio

本地优先的 Windows 文本格式清洗、结构整理与 Word/TXT 导出工具。

**语言：** [English](README.md) · **简体中文** · [繁體中文](README.zh-TW.md) · [日本語](README.ja.md) · [한국어](README.ko.md) · [Español](README.es.md) · [Français](README.fr.md) · [Deutsch](README.de.md) · [Português](README.pt-BR.md) · [Русский](README.ru.md) · [العربية](README.ar.md) · [हिन्दी](README.hi.md)

从 [v1.4.0 Release](https://github.com/SiriZhao/CleanText-Studio/releases/tag/v1.4.0) 下载 Windows 安装版或便携版。v1.4.0 新增 12 种界面语言、运行时切换、阿拉伯语 RTL 与清晰的复选框对勾；本地清洗算法和默认结果保持 v1.3.2 兼容。

更多功能、数学公式、表格、Word 导出和隐私说明见 [English README](README.md)。

# CleanText Studio

CleanText Studio v1.0.0 is a local-first Windows desktop application for deterministic text-format cleanup and polished TXT/DOCX export. It requires no account, API, browser, or network service. It does not provide AI-detection evasion or plagiarism circumvention features.

Run with `python -m cleantext_studio.main`; build instructions are in `docs/BUILD_WINDOWS.md`. MIT licensed.

# CleanText Studio

CleanText Studio v1.1.0 is a local-first Windows desktop application for deterministic text cleanup and TXT/DOCX export. Offline features require no account or API. Optional BYOK AI optimization supports OpenAI, DeepSeek, Anthropic, OpenAI-compatible, and local-compatible endpoints. Selected text is sent only after user confirmation; API keys use Windows Credential Manager. CleanText Studio provides no public API key and does not pay, proxy, or resell model service. Third-party data handling depends on the selected provider. It does not provide AI-detection evasion or plagiarism circumvention features.

Run with `python -m cleantext_studio.main`; build instructions are in `docs/BUILD_WINDOWS.md`. MIT licensed.

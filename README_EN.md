# CleanText Studio

CleanText Studio v1.0.1 is a local-first Windows desktop application for deterministic text-format cleanup and polished TXT/DOCX export. It includes robust level 1–6 Markdown heading cleanup, residual detection, list modes, a modern three-panel UI, and light/dark themes. It requires no account, API, browser, or network service. It does not provide AI-detection evasion or plagiarism circumvention features.

Run with `python -m cleantext_studio.main`; build instructions are in `docs/BUILD_WINDOWS.md`. MIT licensed.
